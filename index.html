Sentiment Analysis on TAPERA Using BERT and BERT-SVM

I conducted sentiment analysis on TAPERA-related tweets using BERT and BERT-SVM. The model classifies sentiments into positive, negative, or neutral, achieving 95% accuracy. Analysis showed 42.54% of 5,912 tweets expressed dissatisfaction with the TAPERA program.

Face Recognition Attendance System

I developed a facial recognition-based attendance system using OpenCV and the face_recognition library in Python. The system captures real-time video, detects faces, matches them with stored data, and logs attendance with a timestamp. It ensures accuracy by calculating face distances and implementing an acceptance threshold.
